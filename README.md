# Carl Joseph P. Nacario — Personal Portfolio

A modern and responsive developer portfolio built using **HTML**, **CSS**, and **Vanilla JavaScript**.  
Designed with a sleek dark interface, animated interactions, and a clean developer-focused aesthetic.

## ✨ Features

- Modern responsive UI/UX
- Animated hero section
- Scroll reveal animations
- Interactive navigation highlighting
- Project showcase cards
- Resume timeline section
- Skills and certifications display
- Contact form validation with toast notifications
- Fully custom CSS design system
- Smooth scrolling and polished transitions

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
- Intersection Observer API
- Responsive Grid & Flexbox Layouts

---

## 🎨 Design Highlights

- Custom CSS variables for theme consistency
- Glassmorphism-inspired navigation
- Floating UI cards and ambient glow effects
- Elegant typography using:
  - Syne
  - DM Sans
  - DM Mono
- Fully handcrafted without frameworks
